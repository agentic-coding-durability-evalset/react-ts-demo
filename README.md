# React TypeScript Demo

A React TypeScript application demo project based on [Create React App](https://create-react-app.dev/). Build traditional React single-page applications using React Scripts and TypeScript.

## Tech Stack

- **React**: 16.14.0
- **TypeScript**: 4.9.5
- **React Scripts**: 5.0.1
- **Fusion Design**: @alifd/next 1.27.32
- **Moment.js**: 2.30.1
- **Testing Library**: Complete testing toolchain

## Project Structure

```
react-ts-demo/
```

## Quick Start

### Prerequisites

- Node.js 14 or higher
- npm or yarn

### Installation and Running

```bash
# Install dependencies
npm install
# Or
yarn install

# Run development server
npm start
# Or
yarn start
```

The application will start at `http://localhost:3000`.

### Build and Test

```bash
# Build production version
npm run build

# Run tests
npm test
```
